# `$ ionic package info`

Get info about a build
## Synopsis

```bash
$ ionic package info [id]
```
  
## Details


Input | Description
----- | ----------
`id` | The build ID. Defaults to the latest build




## Examples

```bash
$ ionic package info 
$ ionic package info 15
```
