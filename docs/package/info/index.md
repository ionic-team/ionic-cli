# package info Command


## Name

package info -- Get info about a build
  
## Synopsis

```bash
$ ionic package info [id]
```
  
## Description

Get info about a build


Input | Description
----- | ----------
`id` | The build ID. Defaults to the latest build




## Examples

```bash
$ ionic package info 
$ ionic package info 15
```
