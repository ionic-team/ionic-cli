# cordova compile Command


## Name

cordova compile -- Compile native platform code
  
## Synopsis

```bash
$ ionic cordova compile <platform>
```
  
## Description

Compile native platform code


Input | Description
----- | ----------
`platform` | The platform to emulate: ios, android




## Examples

```bash
$ ionic cordova compile ios
```
