# `$ ionic cordova prepare`

Transform metadata to platform manifests and copies assets to Cordova platforms
## Synopsis

```bash
$ ionic cordova prepare [platform]
```
  
## Details


Input | Description
----- | ----------
`platform` | The platform you would like to prepare (e.g. ios, android)
