# `$ link`

Connect your local app to Ionic
## Synopsis

```bash
$ ionic link [app_id]
```
  
## Details


Input | Description
----- | ----------
`app_id` | The ID of the app to link (e.g. a1b2c3d4)




## Examples

```bash
$ ionic link 
$ ionic link a1b2c3d4
```
