# `$ ionic upload`

Upload a new snapshot of your app
## Synopsis

```bash
$ ionic upload 
```
  
## Details





Option | Description
------ | ----------
`--note` | Give this snapshot a nice description
`--deploy` | Deploys this snapshot to the given channel
`--nobuild` | Do not invoke a build for this upload

## Examples

```bash
$ ionic upload 
```
